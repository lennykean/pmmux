## Pmmux\.Extensions\.Tls Assembly

TLS termination and SNI\-based routing extension for pmmux\.

### Remarks
Provides TLS/SSL termination and SNI\-based certificate selection\.
Implements `IClientConnectionNegotiator` to intercept connections
and perform TLS handshakes before routing\.
