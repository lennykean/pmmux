## Pmmux\.Core Assembly

Core implementation of the pmmux port multiplexing and routing engine\.

### Remarks
Contains the router, backend monitor, port warden \(UPnP/PMP\), built\-in backends
\(`pass`, `noop`\), and routing strategies \(`first-available`, `least-requests`\)\.
