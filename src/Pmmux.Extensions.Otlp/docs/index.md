## Pmmux\.Extensions\.Otlp Assembly

OpenTelemetry Protocol \(OTLP\) metrics export extension for pmmux\.

### Remarks
Exports telemetry data to OTLP\-compatible backends \(Jaeger, Prometheus, Grafana,
OpenTelemetry Collector\)\. Metrics include connection counts, bytes transferred,
health status, and routing latency\.
